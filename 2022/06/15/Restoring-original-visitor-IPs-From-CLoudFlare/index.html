使用脚本编写Nginx规则恢复CloudFlare原始访问者 IP

CloudFlare 是非常好用的免费 CDN，但是来自 CDN 的请求显示的访问者 IP 将是CDN的 IP ，而不是真实访问者的。Cloudflare 将原始访问者 IP 地址包含在 X-Forwarded-For 和 CF-Connecting-IP 标头中。我们可以通过Nginx来恢复原始访问者的 IP。
